# CircuitCards

A leightweight quiz app about guessing Formula 1 circuits names based on their shape. Built with React Native and Expo.

## Features
- scrollable, rotatable flashcards featuring F1 circuits,
- schematic/shape on the obverse and detailed info on the reverse,
- ability to shuffle the cards.

## Installation

1. **Clone this repository**

   ```bash
   git clone git@github.com:f1shuu/circuitcards.git
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the project**

   ```bash
   npx expo start
   ```

## Download
Download the latest Android APK from the [Releases](https://github.com/f1shuu/circuitcards/releases) page.
